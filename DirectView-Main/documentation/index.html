Prerequisits

You Need These Things To Work with DirectView - Vue Directory Dashboard Template

Node Js

To install and use DirectView, It's required to install Node.js on your computer. Node is required only to run and deploy the application. It's not required to have detailed knowledge of Node.js

Npm

DirectView uses npm package manager to install and manage 3rd party components and libraries.

Installation

Follow the steps below to get started with your Site Template:

First of all it's required to Install Node and npm

You can download DirectView Admin .zip file from Themeforest, Unzip the zip file that you have downloaded from Themeforest. Inside the zip file, you will find the DirectView -main folder and documentation file for documentation.

DirectView uses VueCli for development And NPM To serve and build process

  • 1. Make sure NodeJs installed on your computer.
  • 2. Make sure NPM installed on your computer.
  • 3. Open terminal and navigate to root /source directory and run npm install to install dependencies.

Development server

Run npm run serve to start local development server. Open http://localhost:8080/ in browse.

Build

Run npm run build to build your project. All files including compiled & minified css, js files will be stored in /dist directory.

Grid options

While Bootstrap uses ems or rems for defining most sizes, pxs are used for grid breakpoints and container widths. This is because the viewport width is in pixels and does not change with the font size.
